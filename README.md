# vue-fathom ![npm (scoped)](https://img.shields.io/npm/v/@ubclaunchpad/vue-fathom)

A simple Vue plugin for Fathom Analytics.

```sh
npm i -s @ubclaunchpad/vue-fathom
```

```js
import Vue from 'vue';
import VueFathom from '@ubclaunchpad/vue-fathom';

Vue.use(VueFathom, {
  siteID: 'XXXXXXXX',
  disabled: false,
});
```
